# Google Cluster Trace Analysis with Apache Spark

This project analyzes a 29-day resource trace from a Google cluster consisting of 12,500 machines. Using Apache Spark, we explored how large-scale cloud environments manage hardware, handle maintenance, and allocate resources.

### Project Structure
- google_cluster_analysis.ipynb: The main Jupyter Notebook containing the Spark implementation and data processing logic
- alibaba_cluster_analysis.ipynb: Comparative analysis using the Alibaba 2017 dataset
- report.md: A detailed report discussing our findings, visualizations, and conclusions
